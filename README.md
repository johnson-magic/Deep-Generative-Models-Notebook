> **系统梳理深度生成模型 | Systematic notes and implementations on Deep Generative Models**

## 🎯 项目目标

本项目旨在系统性地整理和实现深度生成模型的核心理论与算法，特别关注：

- **基于分数的生成模型** (Score-based Generative Models)
- **扩散模型** (Diffusion Models)
- 以及它们的理论基础：VAEs、GANs、Score Matching、Denoising Score Matching、Langevin Dynamics、Tweedie's Formula 等

**特点**：
- 📖 理论推导与直观解释并重
- 💻 可执行的 Jupyter Notebook 代码
- 🔗 连接经典统计方法与现代生成模型
- 📚 适合学习者、研究者、实践者

## 🚀 快速开始

### 环境配置

```bash
# 克隆仓库
git clone https://github.com/yourusername/Deep-Generative-Models-Notebook.git
cd Deep-Generative-Models-Notebook

# 安装依赖 (建议使用虚拟环境)
pip install -r requirements.txt
```
